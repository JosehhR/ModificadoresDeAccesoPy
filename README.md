## 🅰️ Conceptos y lectura de codigo

### 1) Selección múltiple

Dada la clase 

```python
class A:
x = 1
_y = 2
__z = 3

a = A()
```
¿Cuáles de los siguientes nombres existen como atributos accesibles directamente desde
a?  
A) a.x  
B) a._y  
C) a.__z  
D) a._A__z
